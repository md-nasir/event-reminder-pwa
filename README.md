# Event Reminder App

## Project setup
```
git clone https://github.com/md-nasir/event-reminder-pwa.git
cd event-reminder-pwa
npm install
```
### Add Api host at .env file
```
VUE_APP_API_BASE_URL=http://127.0.0.1:8000

```

### Compiles and hot-reloads for development
```
npm run serve
```
